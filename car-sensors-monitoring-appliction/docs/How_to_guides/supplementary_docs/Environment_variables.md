## Environment Variables

The `.env` file should contain the following variables:

- **`PORT`**: Port number for the server to listen on.
- **`MONGODB_URI`**: MongoDB connection URI.

Example `.env` file:

```plaintext
PORT=3000
MONGODB_URI=mongodb://localhost:27017/carsensors
```
